# 产品介绍

---

微客红包是借助移动互联网手段,通过扫二维码发红包帮助客户做线下活动,并将客户活动内容在微信朋友圈做精准传播的解决方案。同时为商家提供功能全面的活动制作工具、优惠券营销工具，辅助店铺小程序、拼团电商活动，实现商家的活动、店铺、商品等相关信息的有效传达。
